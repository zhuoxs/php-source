规格设置
开启规格后，商品价格与库存以规格项为准
1. 拖动规格可调整规格显示顺序, 更改规格及规格项后请点击下方的【刷新规格项目表】来更新数据。
2. 每一种规格代表不同型号，例如颜色为一种规格，尺寸为一种规格，如果设置多规格，手机用户必须每一种规格都选择一个规格项，才能进行购买。
3. 设置多规格后，订单的价格，结算，分销金额全部按照核销设置处理。
重要：请勿在商品销售中修改规格参数，刷新规格项目表。
